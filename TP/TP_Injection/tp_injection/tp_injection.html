Préambule BlueJ : import de la librairie femtoContainer

  1. Téléchargez femtoContainer.jar 
  2. Ajouter cette archive à votre projet tp_rules, Menu Outils, item Préférences, onglet Bibliothèques
  3. Redémarrez BlueJ
  4. Vérifiez à nouveau Menu Outils, item Préférences, onglet Bibliothèques, l'archive doit être chargée


Cette archive est un projet bluej autonome, celui-ci contient plusieurs exemples dont quelques patrons de conception ainsi qu'une interface graphique d'aide à l'édition des fichiers de configuration, il peut être ouvert avec le tp_injection.

L'exécution de la méthode main de la classe nommée ConfigurationIHM déclenche une interface graphique qui pourrait vous être utile pour la réalisation des fichiers de configuration ( interface perfectible...)
